# Shopping_316


Ứng dụng web đơn giản để quản lý hàng hóa, có thể chạy trên GitHub Pages.

## Tính năng

- Thêm, chỉnh sửa và xóa các mặt hàng
- Dữ liệu được lưu trữ cục bộ trên trình duyệt
- Giao diện thân thiện, dễ sử dụng
- Responsive, hoạt động tốt trên cả máy tính và di động

## Cách sử dụng

1. Nhấn nút "Thêm Hàng" để thêm mặt hàng mới
2. Nhập thông tin vào các ô tương ứng
3. Nhấn biểu tượng thùng rác để xóa mặt hàng
4. Dữ liệu sẽ tự động được lưu lại
